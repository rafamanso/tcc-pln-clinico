# Classificação Automática de Códigos CID-10 com PLN

Trabalho de Conclusão de Curso — Ciências de Dados e Inteligência Artificial  
Centro Universitário UniDomBosco

## Objetivo

Aplicar técnicas de Processamento de Linguagem Natural (PLN) para classificar 
automaticamente textos clínicos em capítulos do CID-10, utilizando modelos 
Transformer pré-treinados em português brasileiro (BERTimbau).

## Fonte de Dados

Sistema de Informações sobre Mortalidade (SIM/DataSUS) — dados públicos brasileiros.

## Stack

- Python 3.12
- Hugging Face Transformers
- PyTorch
- PySUS
- Pandas, scikit-learn
